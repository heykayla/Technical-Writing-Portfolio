Create a Profile README.md

Overview

A profile README.md is a Markdown file that shares information about yourself and appears on your GitHub profile. In this quickstart guide, youâ€™ll learn how to do the following:

  • Create a repository for your GitHub profile README.md
  • Edit your GitHub profile README.md

This quickstart guide is intended for intermediate users with basic knowledge of Markdown and GitHub.

Part 1: Create a repository

To create a profile README.md file, you must create a repository and name it the same as your GitHub username.

  1. Sign into your GitHub account at https://www.github.com.

    Step Result: After you have signed in, you are directed to the homepage of your GitHub account.

  2. In the navigation menu, click New.
  3. On the Create a New Repository page, do the following:
    1. For the Repository name, enter your GitHub username.
    2. Optional: For the Description, enter a description of your repository.
    3. Keep the default setting to Public.

      Note: Selecting Private makes your profile README.md file publicly inaccessible.

    4. Under Initialize this repository with, select the Add README file checkbox.
    5. Click Create a repository.

Results: The repository and profile README.md file are created.

Part 2: Edit the profile README.md

You can edit the profile README.md to share information about yourself, your work, your skills, and other relevant information.

  1. Locate and open your profile repository.
  2. In the side pane, click Edit README.
  3. When you are finished adding content to your profile README.md file, click Commit changes.

Results: Your profile README.md file is updated and appears in your GitHub profile.

Create a Profile README.md

Overview

In this quick start guide, youâ€™ll create and edit a README.md file for your GitHub profile. A profile README.md is a Markdown file displayed on your GitHub profile that shares information about yourself.

This quick start guide is intended for beginners to intermediate users. It assumes you have basic knowledge of Markdown and GitHub.

Part 1: Create a repository

To create a profile README.md, you must create a repository and name it the same as your username.

  1. Log in to your GitHub account at https://www.github.com. Step Result: After you log in, you are directed to the homepage of your GitHub account.
  2. From the left-hand side navigation, select New.
  3. On the Create a New Repository page, perform the following actions:
    1. Enter your GitHub username in the Repository name field.
    2. Optional: Enter a description of your repository in the Description field.
    3. Keep the default setting to Public. Note:
    4. To create your repository with a README.md, under Initialize this repository with, select Add a README.
    5. To create a repository, select Create a repository.

Part 2: Edit the profile README.md

Once the repository is created, you can edit your profile README.md.

  1. In the repository, on the sidebar, select Edit README.
  2. Populate the README.md file and click Commit changes when finished.

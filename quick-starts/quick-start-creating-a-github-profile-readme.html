Create a Profile README.md

Overview

In this quick start guide, youâ€™ll create and edit a README.md file for your GitHub profile. A profile README.md is a Markdown file displayed on your GitHub profile that shares information about yourself.

This quick start guide is intended for beginners to intermediate users. It assumes you have basic knowledge of Markdown and GitHub.

Part 1: Create a repository

To create a profile README.md, you must create a repository and name it the same as your username.

  1. Log in to your GitHub account at https://www.github.com.

    Step Result: After you log in, you are directed to the homepage of your GitHub account.

  2. From the left-hand side navigation, select New.
  3. On the Create a New Repository page, perform the following actions:
    1. Enter your GitHub username in the Repository name field.
    2. Optional: Enter a description of your repository in the Description field.
    3. Keep the default setting to Public.

      Note: Selecting Private makes your profile README.md file publicly inaccessible.

    4. In the Initialize this repository with section, select the Add README file checkbox.
    5. Select Create a repository.

Results: The repository and profile README.md file are created.

Part 2: Edit the profile README.md

You can edit the profile README.md to share information about yourself, your work, your skills, and other relevant information.

  1. Locate and open your profile repository.
  2. From the right-hand side pane, select Edit README.
  3. When you are finished adding content to your profile README.md file, select Commit changes.

Results: Your profile README.md file displays on your GitHub profile.

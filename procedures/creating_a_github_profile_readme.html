Creating a GitHub Profile README

Introduction

A README file shares information about files in your GitHub repository. You can also create a README for your GitHub profile to share personal information about yourself with other GitHub community members.

In this procedure, you will perform a series of actions for creating a README on your GitHub profile.

Prerequisites

For this procedure, you will need to have a GitHub account. If you do not have a GitHub account, follow GitHub's Signing up for a new GitHub account instructions.

Procedure

  1. Log into your GitHub account at https://www.github.com.
  2. On your GitHub homepage, click + on the top navigation bar and select New repository from the dropdown menu.
  3. On the Create a New Repository page, complete the following actions:
    1. Under the Repository name field, type your GitHub username.
    2. Optional: Under the Description field, you can type a description of your repository (e.g., My personal GitHub repository).
    3. To have your repository be public, select Public.

      NOTE: You need to set your repository to Public; otherwise, users will be unable to view your GitHub profile README.

    4. To create your repository with a README, under Initialize this repository with, select Add a README.
    5. To complete the Create a New Repository page, select Create a repository.
  4. In your repository, on the right-hand sidebar, select Edit README.
  5. Begin editing your GitHub profile README in Markdown.

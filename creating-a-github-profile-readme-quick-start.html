Creating a profile README.md

Overview

In this quick start guide, you’ll create a README.md file for your GitHub profile. A profile README.md is a Markdown file that displays on your GitHub profile, sharing information about your skills, interest, or work. This quick start guide is intended for beginners to intermediate users. It assumes you have basic knowledge of the following:

  • Markdown
  • GitHub

Before you begin

Before you begin this quick start guide, you must have a GitHub account.

Part 1: Create a repository

To create a profile README.md, you must create a repository and name it the same as your username.

  1. Log into your GitHub account at https://www.github.com.
  2. On the navigation bar of the GitHub homepage, click + and select New repository from the dropdown menu.
  3. On the Create a New Repository page, perform the following actions:
    1. Under the Repository name field, enter your GitHub username.
    2. Optional: Under the Description field, enter a description of your repository.
      Example: My personal GitHub repository.
    3. To make your profile README.md publicly accessible, set the repository to public by selecting Public.
    4. To create your repository with a README.md, under Initialize this repository with, select Add a README.
    5. To create a repository, select Create a repository.

Part 2: Edit the profile README.md

Once the repository is created, you can edit your profile README.md.

  1. In the repository, on the sidebar, select Edit README.
  2. Populate the README.md file and click Commit changes when finished.
